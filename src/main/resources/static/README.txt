Для просмотра тестовой страницы работы с вебсокетами:
1) Запустить проект
2) Перейти на localhost:8080

Использование библиотеки: для каждого метода написал описание, стоит взглянуть если не будет работать

Подключение библиотеки:
1) В index.html добавить в <head> <script src="ws/ws-app.js"></script>. Путь стоит перепроверить, в заключительном варианте может лежать в другом месте.
